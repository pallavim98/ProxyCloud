The Web Server:

The web server is a simple base http server running on python using the BaseHttpServer module and its simple functions that help request for pages on the web. The server is being hosted locally on a particular port.

The Proxy Servers:

The web server has 2 proxy servers who are right now being set as slaves to the server to balance the load by choosing them in a sequential + cyclic manner.
With 2 proxies running alternate requests made to the server are run on the two proxies. This can be made more efficeint with more no.of proxies to view the Round Robin Agorithm used clearly.

The Cache:

The Cache has been implemented with a simple single file shared cache system that is common to all proxies and the main webserver itelf. The process goes like this :


Internet Request utility------> HashedCode ------> Compare hash in cache :

If Cache Miss : Save the utility and Use it to open

If Cache Hit : Use the stored utility to open

To do so Hashlib module in python has been implemented with a MD5 algorithm to has the paths/urls of the utilities. Also Urllib module has been made use of to create requests,open urls,set proxies.
